# Context API

## 요약

- 마우스 클릭으로 하단에 있는 박스의 색을 지정합니다.
- 좌/우 클릭으로 좌, 우에 있는 박스의 색을 변경합니다.

## 프로젝트 세팅

- 라이브러리 설치하기

```bash
 npm install --registry https://registry.npmjs.com
```

## 프로젝트 실행하기

```bash
npm run start
```

## 프로젝트 빌드하기

```bash
yarn build
```

## 프로젝트 배포하기

```bash
yarn deploy
```
